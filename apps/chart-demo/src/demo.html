라이브러리 특징

XelStack Chart Library는 함수형 프로그래밍 패러다임을 기반으로 개발된 고성능 차트 라이브러리입니다. 순수 함수와 불변성을 활용하여 예측 가능하고 테스트하기 쉬운 코드를 제공합니다.

주요 특징:

  • 함수형 프로그래밍 패러다임 (74.3% FP 준수율)
  • TypeScript로 작성된 타입 안전성
  • 228개의 테스트 케이스 (100% 통과)
  • 경량 번들 크기 (52KB ESM + CJS)
  • 대용량 데이터 최적화 (10,000+ 포인트)
  • 반응형 & Retina 디스플레이 지원

설치: npm install @xelstack/chart-core
