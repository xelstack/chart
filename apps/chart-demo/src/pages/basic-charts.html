Line Chart 선 그래프

특징

  • 시간에 따른 데이터 변화를 시각화
  • 연속적인 데이터 포인트를 선으로 연결
  • 타이틀과 그리드 표시
  • 커스텀 색상 지원

Bar Chart 막대 그래프

특징

  • 카테고리별 값 비교에 적합
  • 막대 너비 자동 계산
  • 음수 값 지원
  • 그리드를 통한 값 파악 용이

Pie Chart 원형 그래프

특징

  • 전체에서 차지하는 비율 표시
  • 다양한 색상으로 구분
  • 백분율 자동 계산
  • 중앙 정렬 렌더링

Scatter Chart 산점도

특징

  • 두 변수 간의 상관관계 표시
  • 점으로 데이터 포인트 표현
  • 패턴과 추세 파악 용이
  • 대용량 데이터 가상화 지원
