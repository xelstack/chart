실시간 차트 데모

실시간 데이터 업데이트 기능을 확인해보세요.

실시간 선 그래프

새로운 데이터 포인트가 자동으로 추가되는 실시간 차트입니다.

기능: 1초마다 새로운 데이터 포인트가 추가되며, 차트가 부드럽게 업데이트됩니다.

업데이트 횟수: 0
데이터 포인트: 0

실시간 막대 그래프

막대 그래프의 값이 실시간으로 업데이트됩니다.

배치 업데이트

빠른 연속 업데이트가 배치 처리되어 성능을 최적화합니다.

기능: 버튼을 빠르게 클릭하면 여러 업데이트가 배치 처리되어 한 번에 렌더링됩니다.

실시간 스크롤 차트

포인트가 화면에서 움직이면서 다음 차트를 그리는 실시간 스크롤 기능입니다.

기능: 최대 포인트 수를 설정하고 스크롤 방향을 선택할 수 있습니다. 새로운 포인트가 추가되면 기존 포인트들이 설정된 방향으로 이동합니다.

포인트 수: 0
최대 포인트: 100
