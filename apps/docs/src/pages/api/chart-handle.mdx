# ChartHandle

`ChartHandle`은 생성된 차트를 제어하는 인터페이스입니다. `createChart` 함수가 반환하는 객체입니다.

## 인터페이스

```typescript
interface ChartHandle {
  updateData: (dataset: Dataset) => void;
  updateConfig: (config: Partial<ChartConfig>) => void;
  render: () => void;
  destroy: () => void;
  getData: () => Dataset;
  getConfig: () => ChartConfig;
}
```

## 메서드

### updateData

데이터를 업데이트하고 차트를 다시 렌더링합니다.

```typescript
updateData(dataset: Dataset): void
```

**매개변수:**
- `dataset`: 새로운 데이터셋

**예제:**

```javascript
// 전체 데이터 교체
chart.updateData({
  points: [
    { x: 1, y: 100 },
    { x: 2, y: 150 },
    { x: 3, y: 120 }
  ]
});

// 데이터 포인트 추가
const currentData = chart.getData();
chart.updateData({
  points: [...currentData.points, { x: 4, y: 180 }]
});

// 데이터 필터링
const filteredData = {
  points: currentData.points.filter(p => p.y > 100)
};
chart.updateData(filteredData);
```

**참고:**
- 원본 데이터는 변경되지 않습니다 (불변성)
- 업데이트 후 자동으로 `render()`가 호출됩니다

### updateConfig

차트 설정을 업데이트하고 다시 렌더링합니다.

```typescript
updateConfig(config: Partial<ChartConfig>): void
```

**매개변수:**
- `config`: 변경할 설정 옵션 (부분 업데이트 가능)

**예제:**

```javascript
// 제목만 변경
chart.updateConfig({ title: 'New Title' });

// 여러 옵션 동시 변경
chart.updateConfig({
  colors: ['#ff0000', '#00ff00'],
  lineWidth: 3,
  showGrid: false
});

// 차트 타입 변경
chart.updateConfig({ type: 'bar' });
```

**참고:**
- 부분 업데이트를 지원합니다
- 지정하지 않은 옵션은 기존 값 유지
- 업데이트 후 자동으로 `render()`가 호출됩니다

### render

차트를 수동으로 다시 렌더링합니다.

```typescript
render(): void
```

**예제:**

```javascript
// 일반적으로 자동 렌더링되므로 직접 호출 불필요
chart.render();

// 특별한 경우에만 사용 (예: 컨테이너 크기 변경 후)
window.addEventListener('resize', () => {
  chart.render();
});
```

**참고:**
- `updateData()`와 `updateConfig()`는 자동으로 렌더링합니다
- 수동 렌더링이 필요한 경우는 드뭅니다

### destroy

차트를 제거하고 리소스를 정리합니다.

```typescript
destroy(): void
```

**예제:**

```javascript
// 차트 제거
chart.destroy();

// React/Vue에서 컴포넌트 언마운트 시
useEffect(() => {
  const chart = createChart(container, dataset, config);

  return () => {
    chart.destroy();
  };
}, []);
```

**수행 작업:**
- Canvas 요소 제거
- 이벤트 리스너 제거
- ResizeObserver 정리
- 메모리 해제

**참고:**
- destroy 후에는 차트를 사용할 수 없습니다
- 메모리 누수 방지를 위해 반드시 호출하세요

### getData

현재 차트의 데이터를 반환합니다.

```typescript
getData(): Dataset
```

**반환값:**
- 현재 데이터셋의 복사본 (불변)

**예제:**

```javascript
const currentData = chart.getData();
console.log(currentData.points.length);

// 데이터 기반으로 통계 계산
const sum = currentData.points.reduce((acc, p) => acc + p.y, 0);
const avg = sum / currentData.points.length;

// 새 데이터 추가
const newData = {
  points: [...currentData.points, { x: 10, y: 50 }]
};
chart.updateData(newData);
```

**참고:**
- 반환된 데이터를 수정해도 차트에 영향 없음 (불변)
- 업데이트하려면 `updateData()` 사용

### getConfig

현재 차트의 설정을 반환합니다.

```typescript
getConfig(): ChartConfig
```

**반환값:**
- 현재 설정의 복사본 (불변)

**예제:**

```javascript
const currentConfig = chart.getConfig();
console.log(currentConfig.type);
console.log(currentConfig.colors);

// 설정 기반으로 새 설정 생성
const newConfig = {
  ...currentConfig,
  title: 'Updated Title',
  colors: ['#ff0000']
};
chart.updateConfig(newConfig);
```

**참고:**
- 반환된 설정을 수정해도 차트에 영향 없음 (불변)
- 업데이트하려면 `updateConfig()` 사용

## 사용 예제

### 실시간 데이터 업데이트

```javascript
const chart = createChart(container, { points: [] }, config);

setInterval(() => {
  const currentData = chart.getData();
  const newPoint = {
    x: currentData.points.length + 1,
    y: Math.random() * 100
  };

  chart.updateData({
    points: [...currentData.points, newPoint].slice(-50) // 최근 50개만 유지
  });
}, 1000);
```

### 반응형 차트

```javascript
const chart = createChart(container, dataset, config);

// 윈도우 크기 변경 시 차트 크기 조정
const resizeObserver = new ResizeObserver((entries) => {
  const { width, height } = entries[0].contentRect;
  chart.updateConfig({ width, height });
});

resizeObserver.observe(container);

// 정리
chart.destroy();
resizeObserver.disconnect();
```

### 인터랙티브 필터링

```javascript
const chart = createChart(container, dataset, config);

// 필터 버튼 클릭 시
filterButton.addEventListener('click', () => {
  const currentData = chart.getData();
  const filteredData = {
    points: currentData.points.filter(p => p.y > threshold)
  };
  chart.updateData(filteredData);
});
```

### 차트 타입 전환

```javascript
const chart = createChart(container, dataset, { type: 'line' });

toggleButton.addEventListener('click', () => {
  const current = chart.getConfig();
  const newType = current.type === 'line' ? 'bar' : 'line';
  chart.updateConfig({ type: newType });
});
```

### React 통합

```jsx
import { useEffect, useRef } from 'react';
import { createChart } from '@xelstack/chart-core';

function Chart({ data, config }) {
  const containerRef = useRef(null);
  const chartRef = useRef(null);

  useEffect(() => {
    // 차트 생성
    chartRef.current = createChart(
      containerRef.current,
      data,
      config
    );

    // 정리
    return () => {
      chartRef.current?.destroy();
    };
  }, []);

  useEffect(() => {
    // 데이터 업데이트
    chartRef.current?.updateData(data);
  }, [data]);

  useEffect(() => {
    // 설정 업데이트
    chartRef.current?.updateConfig(config);
  }, [config]);

  return <div ref={containerRef} />;
}
```

### Vue 통합

```vue
<template>
  <div ref="chartContainer"></div>
</template>

<script setup>
import { ref, onMounted, onUnmounted, watch } from 'vue';
import { createChart } from '@xelstack/chart-core';

const props = defineProps(['data', 'config']);
const chartContainer = ref(null);
let chart = null;

onMounted(() => {
  chart = createChart(chartContainer.value, props.data, props.config);
});

onUnmounted(() => {
  chart?.destroy();
});

watch(() => props.data, (newData) => {
  chart?.updateData(newData);
});

watch(() => props.config, (newConfig) => {
  chart?.updateConfig(newConfig);
});
</script>
```

## 함수형 패턴

ChartHandle은 함수형 프로그래밍 원칙을 따릅니다:

```javascript
// 1. 불변성 - 원본 데이터 변경 없음
const data1 = { points: [{ x: 1, y: 10 }] };
chart.updateData(data1);
console.log(data1); // 변경 없음

// 2. 순수 함수 - getData/getConfig는 새 객체 반환
const config1 = chart.getConfig();
const config2 = chart.getConfig();
console.log(config1 === config2); // false

// 3. 사이드 이펙트 분리 - render만 DOM 조작
const data = chart.getData(); // 순수 함수
chart.render(); // 사이드 이펙트
```

## 타입 정의

```typescript
import type { ChartHandle, Dataset, ChartConfig } from '@xelstack/chart-core';

// ChartHandle 타입 사용
function updateChart(chart: ChartHandle, newData: Dataset) {
  chart.updateData(newData);
}

// 타입 안전성
const data: Dataset = chart.getData(); // OK
const config: ChartConfig = chart.getConfig(); // OK
```
