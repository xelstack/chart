# createChart

`createChart` 함수는 XelStack Chart의 핵심 API로, 차트를 생성하고 ChartHandle을 반환합니다.

## 함수 시그니처

```typescript
function createChart(
  container: HTMLElement,
  dataset: Dataset,
  config: ChartConfig
): ChartHandle
```

## 매개변수

### container: HTMLElement

차트가 렌더링될 DOM 요소입니다.

```javascript
const container = document.getElementById('my-chart');
const chart = createChart(container, dataset, config);
```

**요구사항:**
- 유효한 HTMLElement여야 합니다
- 너비와 높이가 설정되어 있어야 합니다
- DOM에 마운트되어 있어야 합니다

### dataset: Dataset

차트에 표시할 데이터입니다.

```typescript
interface Dataset {
  points: DataPoint[];
}

interface DataPoint {
  x: number | Date | string;
  y: number;
  series?: string;
}
```

**예제:**

```javascript
// 숫자 데이터
const dataset = {
  points: [
    { x: 1, y: 10 },
    { x: 2, y: 20 },
    { x: 3, y: 15 }
  ]
};

// 날짜 데이터
const dataset = {
  points: [
    { x: new Date('2024-01-01'), y: 100 },
    { x: new Date('2024-01-02'), y: 150 }
  ]
};

// 문자열 데이터 (카테고리)
const dataset = {
  points: [
    { x: 'Jan', y: 30 },
    { x: 'Feb', y: 45 },
    { x: 'Mar', y: 35 }
  ]
};

// 시리즈 데이터 (다중 라인)
const dataset = {
  points: [
    { x: 1, y: 10, series: 'A' },
    { x: 2, y: 20, series: 'A' },
    { x: 1, y: 15, series: 'B' },
    { x: 2, y: 25, series: 'B' }
  ]
};
```

### config: ChartConfig

차트의 설정 옵션입니다.

```typescript
interface ChartConfig {
  type: 'line' | 'bar' | 'pie' | 'scatter';
  width?: number;
  height?: number;
  title?: string;
  colors?: string[];
  showGrid?: boolean;
  lineWidth?: number;
  pointRadius?: number;
  barWidth?: number;
  barGap?: number;
  innerRadius?: number;
  showLabels?: boolean;
  showPercentage?: boolean;
}
```

## 설정 옵션 상세

### 공통 옵션

#### type (필수)
- **타입:** `'line' | 'bar' | 'pie' | 'scatter'`
- **설명:** 차트 타입을 지정합니다

#### width
- **타입:** `number`
- **기본값:** 컨테이너 너비
- **설명:** 차트 너비 (픽셀)

#### height
- **타입:** `number`
- **기본값:** 컨테이너 높이
- **설명:** 차트 높이 (픽셀)

#### title
- **타입:** `string`
- **기본값:** 없음
- **설명:** 차트 제목

#### colors
- **타입:** `string[]`
- **기본값:** `['#3366ff', '#ff3366', '#33ff66', '#ff6633', '#6633ff']`
- **설명:** 차트 색상 배열

#### showGrid
- **타입:** `boolean`
- **기본값:** `true`
- **설명:** 그리드 표시 여부

### Line Chart 옵션

#### lineWidth
- **타입:** `number`
- **기본값:** `2`
- **설명:** 선 두께 (픽셀)

#### pointRadius
- **타입:** `number`
- **기본값:** `3`
- **설명:** 포인트 반지름 (픽셀)

### Bar Chart 옵션

#### barWidth
- **타입:** `number`
- **기본값:** `0.8` (상대값)
- **설명:** 막대 너비

#### barGap
- **타입:** `number`
- **기본값:** `0.2` (상대값)
- **설명:** 막대 간격

### Pie Chart 옵션

#### innerRadius
- **타입:** `number`
- **기본값:** `0`
- **설명:** 내부 반지름 (도넛 차트)

#### showLabels
- **타입:** `boolean`
- **기본값:** `false`
- **설명:** 라벨 표시 여부

#### showPercentage
- **타입:** `boolean`
- **기본값:** `false`
- **설명:** 퍼센트 표시 여부

### Scatter Chart 옵션

#### pointRadius
- **타입:** `number`
- **기본값:** `3`
- **설명:** 포인트 반지름 (픽셀)

## 반환값

[ChartHandle](/api/chart-handle) 객체를 반환합니다.

## 사용 예제

### 기본 사용

```javascript
import { createChart } from '@xelstack/chart-core';

const container = document.getElementById('chart');
const dataset = {
  points: [
    { x: 1, y: 10 },
    { x: 2, y: 20 },
    { x: 3, y: 15 }
  ]
};

const config = {
  type: 'line',
  title: 'Sales Data',
  showGrid: true
};

const chart = createChart(container, dataset, config);
```

### 커스텀 스타일링

```javascript
const config = {
  type: 'line',
  title: 'Custom Styled Chart',
  width: 1000,
  height: 500,
  colors: ['#ff6b6b', '#4ecdc4', '#45b7d1'],
  lineWidth: 3,
  pointRadius: 5,
  showGrid: true
};

const chart = createChart(container, dataset, config);
```

### 막대 차트

```javascript
const config = {
  type: 'bar',
  title: 'Monthly Revenue',
  colors: ['#3366ff'],
  barWidth: 0.7,
  barGap: 0.3
};

const chart = createChart(container, dataset, config);
```

### 도넛 차트

```javascript
const config = {
  type: 'pie',
  title: 'Market Share',
  innerRadius: 50,
  colors: ['#ff6384', '#36a2eb', '#ffce56']
};

const chart = createChart(container, dataset, config);
```

## 에러 처리

```javascript
try {
  const chart = createChart(container, dataset, config);
} catch (error) {
  console.error('Failed to create chart:', error);
}
```

### 일반적인 에러

- **Invalid container**: 컨테이너가 null이거나 유효하지 않음
- **Invalid dataset**: 데이터셋이 비어있거나 형식이 잘못됨
- **Invalid config**: 필수 설정(type)이 누락되었거나 잘못됨

## 함수형 프로그래밍

`createChart`는 순수 함수형 API를 제공합니다:

```javascript
// 사이드 이펙트 분리
const chart = createChart(container, dataset, config);

// 불변 데이터 업데이트
const newDataset = {
  points: [...dataset.points, { x: 4, y: 25 }]
};
chart.updateData(newDataset);

// 원본 데이터는 변경되지 않음
console.log(dataset.points.length); // 여전히 3
```
