## 핵심 API

### createChart

차트를 생성하고 ChartHandle을 반환하는 핵심 함수입니다.

```typescript
function createChart(
  container: HTMLElement,
  dataset: Dataset,
  config: ChartConfig
): ChartHandle
```

**매개변수:**
- `container`: 차트를 렌더링할 HTML 요소
- `dataset`: 차트 데이터 (`{ points: DataPoint[] }`)
- `config`: 차트 설정 (type, width, height 등)

**반환값:**
`ChartHandle` - 차트를 제어할 수 있는 핸들 객체

**예제:**

```typescript
import { createChart } from '@xelstack/chart-core';

const chart = createChart(
  document.getElementById('chart'),
  { points: [{ x: 1, y: 10 }, { x: 2, y: 20 }] },
  { type: 'line', title: 'My Chart', showGrid: true }
);
```

**자세한 문서:** [createChart API](/api/create-chart)

### ChartHandle

차트를 제어하는 인터페이스입니다.

```typescript
interface ChartHandle {
  updateData: (dataset: Dataset) => void;
  updateConfig: (config: Partial<ChartConfig>) => void;
  render: () => void;
  destroy: () => void;
  getData: () => Dataset;
  getConfig: () => ChartConfig;
}
```

**메서드:**
- `updateData(dataset)` - 데이터 업데이트 후 자동 재렌더링
- `updateConfig(config)` - 설정 업데이트 후 자동 재렌더링
- `render()` - 수동 재렌더링
- `destroy()` - 차트 제거 및 리소스 정리
- `getData()` - 현재 데이터 반환 (불변)
- `getConfig()` - 현재 설정 반환 (불변)

**자세한 문서:** [ChartHandle API](/api/chart-handle)

## 순수 함수

순수 함수는 동일한 입력에 대해 항상 동일한 출력을 반환하며, 부수 효과가 없습니다.

### 뷰포트 계산

#### calculateViewportFromDataset

데이터셋으로부터 적절한 뷰포트를 자동 계산합니다.

```typescript
function calculateViewportFromDataset(
  dataset: Dataset,
  padding?: number
): Viewport
```

**예제:**

```typescript
import { calculateViewportFromDataset } from '@xelstack/chart-core';

const viewport = calculateViewportFromDataset(dataset, 0.1);
// { xMin: 0, xMax: 110, yMin: 0, yMax: 55, zoomLevel: 1.0 }
```

#### calculateZoomedViewport

줌 레벨을 적용한 뷰포트를 계산합니다.

```typescript
function calculateZoomedViewport(
  viewport: Viewport,
  zoomLevel: number,
  centerX: number,
  centerY: number
): Viewport
```

#### calculatePannedViewport

팬(이동)을 적용한 뷰포트를 계산합니다.

```typescript
function calculatePannedViewport(
  viewport: Viewport,
  deltaX: number,
  deltaY: number
): Viewport
```

```typescript
function deepMergeConfig(

): ChartConfig
```

### 데이터 변환

#### filterPoints

조건에 맞는 데이터 포인트만 필터링합니다.

```typescript
function filterPoints(
  dataset: Dataset,
  predicate: (point: DataPoint) => boolean
): Dataset
```

**예제:**

```typescript
import { filterPoints } from '@xelstack/chart-core';

const filtered = filterPoints(dataset, (p) => p.y > 20);
```

#### sortPoints

데이터 포인트를 정렬합니다.

```typescript
function sortPoints(
  dataset: Dataset,
  compareFn: (a: DataPoint, b: DataPoint) => number
): Dataset
```

**예제:**

```typescript
import { sortPoints } from '@xelstack/chart-core';

const sorted = sortPoints(dataset, (a, b) => a.x - b.x);
```

#### mapPoints

각 데이터 포인트를 변환합니다.

```typescript
function mapPoints(
  dataset: Dataset,
  mapper: (point: DataPoint) => DataPoint
): Dataset
```

**예제:**

```typescript
import { mapPoints } from '@xelstack/chart-core';

const doubled = mapPoints(dataset, (p) => ({ ...p, y: p.y * 2 }));
```

## 함수형 유틸리티

### 함수 합성

#### pipe

함수들을 왼쪽에서 오른쪽으로 합성합니다.

```typescript
function pipe<T, R>(...fns: Array<(input: any) => any>): (input: T) => R
```

**예제:**

```typescript
import { pipe, filterPoints, sortPoints, mapPoints } from '@xelstack/chart-core';

const transform = pipe(
  (data) => filterPoints(data, (p) => p.y > 0),
  (data) => sortPoints(data, (a, b) => a.x - b.x),
  (data) => mapPoints(data, (p) => ({ ...p, y: p.y * 2 }))
);

const result = transform(dataset);
```

#### compose

함수들을 오른쪽에서 왼쪽으로 합성합니다.

```typescript
function compose<T, R>(...fns: Array<(input: any) => any>): (input: T) => R
```

### 커링

#### curry

함수를 커링된 형태로 변환합니다.

```typescript
function curry<T extends any[], R>(fn: (...args: T) => R): Curried<T, R>
```

**예제:**

```typescript
import { curry } from '@xelstack/chart-core';

const add = (a: number, b: number, c: number) => a + b + c;
const curriedAdd = curry(add);

console.log(curriedAdd(5)(10)(3)); // 18
```

### 불변성 유틸리티

#### deepFreeze

객체를 깊게 동결합니다.

```typescript
function deepFreeze<T>(obj: T): Readonly<T>
```

#### immutableUpdate

불변 방식으로 객체를 업데이트합니다.

```typescript
function immutableUpdate<T>(obj: T, updates: Partial<T>): T
```

## 타입 정의

전체 타입 정의는 [Types API 문서](/api/types)를 참조하세요.

### 기본 타입

```typescript
// 데이터 포인트
interface DataPoint {
  x: number | Date | string;
  y: number;
  series?: string;
}

// 데이터셋
interface Dataset {
  points: DataPoint[];
}

// 차트 타입
type ChartType = 'line' | 'bar' | 'pie' | 'scatter';

// 차트 설정
interface ChartConfig {
  type: ChartType;
  width?: number;
  height?: number;
  title?: string;
  colors?: string[];
  showGrid?: boolean;
  lineWidth?: number;
  pointRadius?: number;
  barWidth?: number;
  barGap?: number;
  innerRadius?: number;
}

// 뷰포트
interface Viewport {
  xMin: number;
  xMax: number;
  yMin: number;
  yMax: number;
  zoomLevel: number;
}
```

## 에러 클래스

### ValidationError

데이터 검증 실패 시 발생하는 에러입니다.

```typescript
class ValidationError extends Error {
  constructor(message: string);
}
```

### ChartTypeError

잘못된 차트 타입 사용 시 발생하는 에러입니다.

```typescript
class ChartTypeError extends Error {
  constructor(type: string);
}
```

**예제:**

```typescript
import { createChart, ValidationError } from '@xelstack/chart-core';

try {
  createChart(container, invalidDataset, config);
} catch (error) {
  if (error instanceof ValidationError) {
    console.error('데이터 검증 실패:', error.message);
  }
}
```
