# 차트 타입

XelStack Chart는 다양한 차트 타입을 지원합니다. 각 차트 타입은 특정 데이터 시각화 목적에 최적화되어 있습니다.

## 지원하는 차트 타입

### Line Chart (선 차트)

시간에 따른 데이터 변화를 표시하는 데 적합합니다.

```typescript
const config = {
  type: 'line',
  title: '월별 매출 추이',
};
```

**사용 사례:**
- 시계열 데이터 (매출, 주가, 온도 등)
- 트렌드 분석
- 연속적인 데이터 변화

[상세 가이드 보기](/charts/line)

---

### Bar Chart (막대 차트)

카테고리별 데이터 비교에 적합합니다.

```typescript
const config = {
  type: 'bar',
  title: '분기별 실적',
};
```

**사용 사례:**
- 카테고리 간 비교
- 순위 데이터
- 설문조사 결과

[상세 가이드 보기](/charts/bar)

---

### Pie Chart (파이 차트)

전체 대비 부분의 비율을 표시하는 데 적합합니다.

```typescript
const config = {
  type: 'pie',
  title: '시장 점유율',
};
```

**사용 사례:**
- 비율/점유율 표시
- 예산 분배
- 설문 결과 (선택지별 비중)

[상세 가이드 보기](/charts/pie)

---

### Scatter Chart (산점도)

두 변수 간의 관계를 표시하는 데 적합합니다.

```typescript
const config = {
  type: 'scatter',
  title: '키와 몸무게 분포',
};
```

**사용 사례:**
- 상관관계 분석
- 분포 패턴 파악
- 이상치 탐지

[상세 가이드 보기](/charts/scatter)

## 차트 타입 선택 가이드

### 데이터 유형별 권장 차트

| 데이터 유형 | 권장 차트 타입 | 이유 |
|------------|----------------|------|
| 시계열 데이터 | Line Chart | 시간에 따른 변화를 명확히 표현 |
| 카테고리 비교 | Bar Chart | 각 항목 간 크기 비교가 쉬움 |
| 비율/구성 | Pie Chart | 전체 대비 부분의 비중을 직관적으로 표현 |
| 두 변수 관계 | Scatter Chart | 상관관계와 분포를 시각화 |

### 데이터 크기별 권장 사항

| 데이터 크기 | Line | Bar | Pie | Scatter |
|------------|------|-----|-----|---------|
| 소량 (< 10개) | ✓ | ✓ | ✓ | ✓ |
| 중량 (10-100개) | ✓ | ✓ | ✗ | ✓ |
| 대량 (100-1000개) | ✓ | △ | ✗ | ✓ |
| 초대량 (> 1000개) | ✓ | ✗ | ✗ | ✓ |

- ✓ 권장
- △ 조건부 권장
- ✗ 비권장

## 공통 설정

모든 차트 타입에서 사용할 수 있는 공통 설정입니다.

### 크기 설정

```typescript
const config = {
  type: 'line',
  width: 800,   // 픽셀 단위
  height: 600,  // 픽셀 단위
};
```

### 제목 설정

```typescript
const config = {
  type: 'line',
  title: '내 차트 제목',
};
```

### 색상 설정

```typescript
const config = {
  type: 'line',
  colors: ['#3366ff', '#ff6633', '#33ff66'],
};
```

### 그리드 표시

```typescript
const config = {
  type: 'line',
  showGrid: true,
};
```

## 차트 타입 전환

동일한 데이터로 차트 타입을 쉽게 전환할 수 있습니다.

```typescript
const dataset = {
  points: [
    { x: '1월', y: 100 },
    { x: '2월', y: 150 },
    { x: '3월', y: 200 },
  ],
};

// Line 차트로 시작
const chart = createChart(container, dataset, {
  type: 'line',
});

// Bar 차트로 전환
chart.updateConfig({
  type: 'bar',
});
chart.render();

// Pie 차트로 전환
chart.updateConfig({
  type: 'pie',
});
chart.render();
```

## 성능 최적화

### 대용량 데이터

각 차트 타입은 대용량 데이터를 자동으로 최적화합니다:

- **Line Chart**: 10,000개 이상 데이터 포인트 자동 샘플링
- **Bar Chart**: 가상 스크롤링으로 수천 개 막대 처리
- **Scatter Chart**: 캔버스 기반 렌더링으로 수만 개 포인트 지원
- **Pie Chart**: 자동 그룹핑 (작은 비율 항목 통합)

### 반응형 디자인

```typescript
// 컨테이너 크기에 맞춤
const chart = createChart(container, dataset, {
  type: 'line',
  width: container.clientWidth,
  height: container.clientHeight,
});

// 리사이즈 이벤트 처리
window.addEventListener('resize', () => {
  chart.updateConfig({
    width: container.clientWidth,
    height: container.clientHeight,
  });
  chart.render();
});
```
