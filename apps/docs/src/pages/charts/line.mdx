# Line Chart (선 차트)

선 차트는 시간에 따른 데이터 변화를 표시하는 데 가장 적합한 차트입니다. 연속적인 데이터의 트렌드를 파악하기 쉽습니다.

## 기본 사용법

```typescript
import { createChart } from '@xelstack/chart-core';

const dataset = {
  points: [
    { x: 0, y: 10 },
    { x: 1, y: 20 },
    { x: 2, y: 15 },
    { x: 3, y: 25 },
    { x: 4, y: 22 },
  ],
};

const chart = createChart(container, dataset, {
  type: 'line',
  title: '월별 매출 추이',
  width: 800,
  height: 600,
  colors: ['#3366ff'],
  showGrid: true,
});
```

## 고급 기능

### 여러 라인 표시

```typescript
const multiSeriesData = {
  points: [
    // 시리즈 A
    { x: 0, y: 10, series: 'A' },
    { x: 1, y: 20, series: 'A' },
    { x: 2, y: 15, series: 'A' },
    // 시리즈 B
    { x: 0, y: 15, series: 'B' },
    { x: 1, y: 25, series: 'B' },
    { x: 2, y: 20, series: 'B' },
  ],
};

const chart = createChart(container, multiSeriesData, {
  type: 'line',
  title: '제품별 매출 비교',
  colors: ['#3366ff', '#ff6633'], // 시리즈별 색상
});
```

### 시간 데이터

### 대용량 데이터

```typescript
// 10,000개 데이터 포인트
const largeDataset = {
  points: Array.from({ length: 10000 }, (_, i) => ({
    x: i,
    y: Math.sin(i / 100) * 100 + Math.random() * 20,
  })),
};

// 자동 샘플링 적용

  type: 'line',
  title: '고빈도 센서 데이터',
});
```

## 스타일링

### 라인 색상

```typescript
const config = {
  type: 'line',
  colors: [
    '#3366ff', // 첫 번째 라인
    '#ff6633', // 두 번째 라인
    '#33ff66', // 세 번째 라인
  ],
};
```

### 그리드 표시

```typescript
const config = {
  type: 'line',
  showGrid: true, // X, Y축 그리드 표시
};
```

## 실시간 업데이트

```typescript
const chart = createChart(container, initialData, config);

// 1초마다 새 데이터 추가
setInterval(() => {
  const newPoint = {
    x: Date.now(),
    y: Math.random() * 100,
  };

  // 마지막 100개 포인트만 유지
  const updatedPoints = [...currentData.points, newPoint].slice(-100);

  chart.updateData({ points: updatedPoints });
  chart.render();
}, 1000);
```

## 성능 최적화

### 자동 샘플링

10,000개 이상의 데이터 포인트는 자동으로 샘플링되어 성능이 최적화됩니다.

```typescript
// 100,000개 데이터 - 자동 샘플링
const massiveData = {
  points: Array.from({ length: 100000 }, (_, i) => ({
    x: i,
    y: Math.random() * 100,
  })),
};

// 렌더링 성능 유지
const chart = createChart(container, massiveData, {
  type: 'line',
});
```

### 수동 샘플링

필요시 직접 샘플링할 수도 있습니다.

```typescript
// 10개 중 1개만 표시
const sampledData = {
  points: rawData.points.filter((_, i) => i % 10 === 0),
};
```

## 접근성

Line Chart는 다음 접근성 기능을 제공합니다:

- 키보드 네비게이션 지원
- 스크린 리더 호환
- 고대비 모드 지원
