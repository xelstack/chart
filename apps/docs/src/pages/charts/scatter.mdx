# Scatter Chart (산점도)

산점도는 두 변수 간의 관계를 표시하는 데 가장 효과적인 차트입니다. 상관관계, 분포 패턴, 이상치를 쉽게 파악할 수 있습니다.

## 기본 사용법

```typescript
import { createChart } from '@xelstack/chart-core';

const dataset = {
  points: [
    { x: 160, y: 55 },
    { x: 165, y: 60 },
    { x: 170, y: 65 },
    { x: 175, y: 70 },
    { x: 180, y: 75 },
  ],
};

const chart = createChart(container, dataset, {
  type: 'scatter',
  title: '키와 몸무게의 관계',
  width: 800,
  height: 600,
  colors: ['#3366ff'],
  showGrid: true,
});
```

## 고급 기능

### 여러 그룹 표시

```typescript
const multiGroupData = {
  points: [
    // 그룹 A
    { x: 160, y: 55, series: '남성' },
    { x: 165, y: 60, series: '남성' },
    { x: 170, y: 65, series: '남성' },
    // 그룹 B
    { x: 155, y: 50, series: '여성' },
    { x: 160, y: 53, series: '여성' },
    { x: 165, y: 57, series: '여성' },
  ],
};

const chart = createChart(container, multiGroupData, {
  type: 'scatter',
  title: '성별 키-몸무게 분포',
  colors: ['#3366ff', '#ff6633'],
});
```

### 포인트 크기 조정

```typescript
const sizedData = {
  points: [
    { x: 100, y: 200, size: 10 },
    { x: 150, y: 250, size: 20 },
    { x: 200, y: 300, size: 30 },
  ],
};

const chart = createChart(container, sizedData, {
  type: 'scatter',
  pointSize: 'dynamic', // size 속성 기반
});
```

## 스타일링

### 포인트 색상

```typescript
const config = {
  type: 'scatter',
  colors: [
    '#3366ff', // 첫 번째 그룹
    '#ff6633', // 두 번째 그룹
    '#33ff66', // 세 번째 그룹
  ],
};
```

### 포인트 크기

```typescript
const config = {
  type: 'scatter',
  pointSize: 8, // 고정 크기 (픽셀)
};
```

## 사용 사례

### 상관관계 분석

```typescript
const correlationData = {
  points: Array.from({ length: 100 }, () => {
    const x = Math.random() * 100;
    const y = x * 2 + Math.random() * 20; // 양의 상관관계
    return { x, y };
  }),
};

const chart = createChart(container, correlationData, {
  type: 'scatter',
  title: '광고비와 매출의 상관관계',
  showGrid: true,
});
```

### 클러스터 분석

```typescript
const clusterData = {
  points: [
    // 클러스터 1
    ...Array.from({ length: 30 }, () => ({
      x: 20 + Math.random() * 10,
      y: 30 + Math.random() * 10,
      series: 'A',
    })),
    // 클러스터 2
    ...Array.from({ length: 30 }, () => ({
      x: 50 + Math.random() * 10,
      y: 60 + Math.random() * 10,
      series: 'B',
    })),
  ],
};

const chart = createChart(container, clusterData, {
  type: 'scatter',
  title: '고객 세그먼트 분석',
  colors: ['#3366ff', '#ff6633'],
});
```

### 이상치 탐지

```typescript
const outlierData = {
  points: [
    // 정상 데이터
    ...Array.from({ length: 95 }, () => ({
      x: 50 + Math.random() * 20,
      y: 50 + Math.random() * 20,
      series: '정상',
    })),
    // 이상치
    { x: 90, y: 10, series: '이상치' },
    { x: 10, y: 90, series: '이상치' },
    { x: 95, y: 95, series: '이상치' },
  ],
};

const chart = createChart(container, outlierData, {
  type: 'scatter',
  title: '데이터 이상치 탐지',
  colors: ['#3366ff', '#ff0000'],
});
```

## 성능 최적화

### 대용량 데이터

산점도는 캔버스 기반 렌더링으로 수만 개의 포인트를 빠르게 처리합니다.

```typescript
// 50,000개 데이터 포인트
const massiveData = {
  points: Array.from({ length: 50000 }, () => ({
    x: Math.random() * 1000,
    y: Math.random() * 1000,
  })),
};

// 여전히 빠른 렌더링
const chart = createChart(container, massiveData, {
  type: 'scatter',
  title: '대규모 센서 데이터',
});
```

## 접근성

Scatter Chart는 다음 접근성 기능을 제공합니다:

- 툴팁으로 정확한 값 표시
- 색각 이상자를 위한 다양한 색상 팔레트
- 고대비 모드 지원
