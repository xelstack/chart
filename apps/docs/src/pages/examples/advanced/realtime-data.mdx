# Realtime Data

실시간으로 업데이트되는 데이터를 효율적으로 처리하는 방법입니다.

## 증분 렌더링 (권장)

증분 렌더링은 새 데이터만 렌더링하여 성능을 93% 이상 향상시킵니다. 기존 방식 대비 10,000개 데이터에 10개 추가 시 ~30ms → ~2ms로 단축됩니다.

### 기본 사용법

```typescript
import { createChart } from '@xelstack/chart-core';

const chart = createChart(container, { points: [] }, {
  type: 'line',
  title: 'Realtime Chart (Incremental)',
  width: 800,
  height: 400
});

// 최대 포인트 수 설정 (슬라이딩 윈도우)
chart.setIncrementalOptions({ maxPoints: 500 });

// 실시간 데이터 추가 - 증분 렌더링
setInterval(() => {
  chart.addPointsIncremental([{
    x: Date.now(),
    y: Math.random() * 100
  }]);
}, 100);
```

### 배치 업데이트

외부에서 전체 데이터를 받을 때 델타만 계산하여 렌더링:

```typescript
// 서버에서 전체 데이터 수신 시
async function syncWithServer() {
  const serverData = await fetch('/api/data').then(r => r.json());

  // 자동으로 변경분만 계산하여 렌더링
  chart.updateDataIncremental(serverData.points);
}
```

```typescript
// 탭 비활성화 시 일시정지
document.addEventListener('visibilitychange', () => {
```

### WebSocket + 증분 렌더링

```typescript
const chart = createChart(container, { points: [] }, config);
chart.setIncrementalOptions({ maxPoints: 1000 });

const ws = new WebSocket('ws://localhost:8080/data');

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);

  // 증분 렌더링 - 내부적으로 프레임 버퍼링
  chart.addPointsIncremental([{
    x: data.timestamp,
    y: data.value
  }]);
};

ws.onclose = () => chart.pauseIncremental();
ws.onopen = () => chart.resumeIncremental();
```

## 기존 방식

아래는 기존 방식의 실시간 차트 구현입니다. 증분 렌더링을 사용할 수 없는 환경에서 참고하세요.

## 기본 실시간 차트

```typescript
import { createChart } from '@xelstack/chart-core';

const chart = createChart(container, { points: [] }, {
  type: 'line',
  title: 'Realtime Chart',
  width: 800,
  height: 400,
  showGrid: true
});

// 1초마다 새 데이터 추가
setInterval(() => {
  const currentData = chart.getData();
  const newPoint = {
    x: Date.now(),
    y: Math.random() * 100
  };

  chart.updateData({
    points: [...currentData.points, newPoint]
  });
  chart.render();
}, 1000);
```

## 슬라이딩 윈도우

최근 N개의 데이터만 유지:

```typescript
const MAX_POINTS = 50;

function addRealtimeData(chart, newPoint) {
  const currentData = chart.getData();
  const newPoints = [...currentData.points, newPoint].slice(-MAX_POINTS);

  chart.updateData({ points: newPoints });
  chart.render();
}

// 사용
setInterval(() => {
  addRealtimeData(chart, {
    x: Date.now(),
    y: Math.random() * 100
  });
}, 100);
```

## 다중 시리즈 실시간 데이터

```typescript
function addMultiSeriesData(chart, newValues) {
  const currentData = chart.getData();
  const timestamp = Date.now();

  const newPoints = Object.entries(newValues).map(([series, value]) => ({
    x: timestamp,
    y: value,
    series
  }));

  const allPoints = [...currentData.points, ...newPoints];

  // 각 시리즈별로 최근 50개만 유지
  const filteredPoints = {};
  allPoints.forEach(p => {
    const key = p.series || 'default';
    if (!filteredPoints[key]) {
      filteredPoints[key] = [];
    }
    filteredPoints[key].push(p);
  });

  const finalPoints = Object.values(filteredPoints).flatMap(points =>
    points.slice(-MAX_POINTS)
  );

  chart.updateData({ points: finalPoints });
  chart.render();
}

// 사용
setInterval(() => {
  addMultiSeriesData(chart, {
    'CPU': Math.random() * 100,
    'Memory': Math.random() * 100,
    'Network': Math.random() * 100
  });
}, 1000);
```

## WebSocket 연결

```typescript
class RealtimeChart {
  constructor(container, config, wsUrl) {
    this.chart = createChart(container, { points: [] }, config);
    this.buffer = [];
    this.maxPoints = 100;
    this.setupWebSocket(wsUrl);
    this.startRenderLoop();
  }

  setupWebSocket(url) {
    this.ws = new WebSocket(url);

    this.ws.onmessage = (event) => {
      const data = JSON.parse(event.data);
      this.buffer.push({
        x: data.timestamp,
        y: data.value,
        series: data.series
      });
    };

    this.ws.onerror = (error) => {
      console.error('WebSocket error:', error);
    };

    this.ws.onclose = () => {
      console.log('WebSocket closed, attempting to reconnect...');
      setTimeout(() => this.setupWebSocket(url), 5000);
    };
  }

  startRenderLoop() {
    // 60 FPS 렌더링
    const render = () => {
      if (this.buffer.length > 0) {
        this.flushBuffer();
      }
      requestAnimationFrame(render);
    };
    requestAnimationFrame(render);
  }

  flushBuffer() {
    const currentData = this.chart.getData();
    const newPoints = [...currentData.points, ...this.buffer]
      .slice(-this.maxPoints);

    this.chart.updateData({ points: newPoints });
    this.chart.render();

    this.buffer = [];
  }

  destroy() {
    if (this.ws) {
      this.ws.close();
    }
    this.chart.destroy();
  }
}

// 사용
const realtimeChart = new RealtimeChart(
  document.getElementById('chart'),
  { type: 'line', title: 'WebSocket Stream' },
  'ws://localhost:8080/data'
);
```

## 성능 최적화

배치 업데이트와 쓰로틀링:

```typescript
class OptimizedRealtimeChart {
  constructor(container, config) {
    this.chart = createChart(container, { points: [] }, config);
    this.buffer = [];
    this.updateInterval = 100; // ms
    this.maxPoints = 200;

    this.startUpdateLoop();
  }

  addData(point) {
    this.buffer.push(point);
  }

  startUpdateLoop() {
    setInterval(() => {
      if (this.buffer.length > 0) {
        this.flushBuffer();
      }
    }, this.updateInterval);
  }

  flushBuffer() {
    const currentData = this.chart.getData();
    const newPoints = [...currentData.points, ...this.buffer]
      .slice(-this.maxPoints);

    this.chart.updateData({ points: newPoints });
    this.chart.render();

    this.buffer = [];
  }
}

// 사용 - 매우 빠른 데이터 스트림
const chart = new OptimizedRealtimeChart(
  container,
  { type: 'line', title: 'High-Speed Stream' }
);

// 데이터가 빠르게 들어와도 100ms마다 한 번만 업데이트
setInterval(() => {
  chart.addData({
    x: Date.now(),
    y: Math.random() * 100
  });
}, 10); // 10ms마다 데이터 추가 (100 FPS)
```

## 시간축 자동 스크롤

```typescript
function createScrollingChart(container, config) {
  const chart = createChart(container, { points: [] }, config);
  const timeWindow = 60000; // 60초 윈도우

  function addData(point) {
    const currentData = chart.getData();
    const now = Date.now();

    // 시간 윈도우 내의 포인트만 유지
    const filteredPoints = currentData.points.filter(p =>
      now - p.x < timeWindow
    );

    chart.updateData({
      points: [...filteredPoints, point]
    });

    // 뷰포트 업데이트 (시간축 스크롤)
    chart.updateConfig({
      viewport: {
        xMin: now - timeWindow,
        xMax: now,
        yMin: 0,
        yMax: 100
      }
    });

    chart.render();
  }

  return { chart, addData };
}
```

## 완전한 예제

```typescript
class AdvancedRealtimeChart {
  constructor(container, config, options = {}) {
    this.maxPoints = options.maxPoints || 100;
    this.updateInterval = options.updateInterval || 100;
    this.timeWindow = options.timeWindow || 60000;

    this.chart = createChart(container, { points: [] }, config);
    this.buffer = [];
    this.isPaused = false;
    this.stats = { fps: 0, dataRate: 0 };

    this.startUpdateLoop();
    this.startStatsLoop();
  }

  addData(point) {
    if (!this.isPaused) {
      this.buffer.push({
        ...point,
        x: point.x || Date.now()
      });
    }
  }

  startUpdateLoop() {
    setInterval(() => {
      if (!this.isPaused && this.buffer.length > 0) {
        this.flushBuffer();
      }
    }, this.updateInterval);
  }

  flushBuffer() {
    const currentData = this.chart.getData();
    const now = Date.now();

    // 시간 윈도우 필터링
    const timeFilteredPoints = currentData.points.filter(p =>
      now - p.x < this.timeWindow
    );

    // 새 포인트 추가
    const newPoints = [...timeFilteredPoints, ...this.buffer]
      .slice(-this.maxPoints);

    this.chart.updateData({ points: newPoints });
    this.chart.render();

    this.stats.dataRate = this.buffer.length;
    this.buffer = [];
  }

  startStatsLoop() {
    let frameCount = 0;
    let lastTime = Date.now();

    const updateStats = () => {
      frameCount++;
      const now = Date.now();

      if (now - lastTime >= 1000) {
        this.stats.fps = frameCount;
        frameCount = 0;
        lastTime = now;

        this.displayStats();
      }

      requestAnimationFrame(updateStats);
    };

    requestAnimationFrame(updateStats);
  }

  displayStats() {
    console.log(`FPS: ${this.stats.fps}, Data Rate: ${this.stats.dataRate}/s`);
  }

  pause() {
    this.isPaused = true;
  }

  resume() {
    this.isPaused = false;
  }

  clear() {
    this.chart.updateData({ points: [] });
    this.buffer = [];
    this.chart.render();
  }

  destroy() {
    this.chart.destroy();
  }
}

// 사용
const chart = new AdvancedRealtimeChart(
  document.getElementById('chart'),
  {
    type: 'line',
    title: 'Advanced Realtime Chart',
    showGrid: true
  },
  {
    maxPoints: 200,
    updateInterval: 50,
    timeWindow: 30000
  }
);

// 데이터 스트림 시뮬레이션
setInterval(() => {
  chart.addData({
    y: Math.random() * 100
  });
}, 50);
```

| 시나리오 | 기존 방식 | 증분 렌더링 | 개선율 |
