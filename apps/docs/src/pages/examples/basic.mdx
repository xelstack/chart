# Basic Examples (기본 예제)

가장 기본적인 차트 사용 예제를 통해 XelStack Chart의 핵심 기능을 익혀보세요.

## Line Chart (선 차트)

### 기본 선 차트

가장 간단한 선 차트 예제입니다.

<ChartExample
  title="Simple Line Chart"
  description="월별 매출 데이터를 선 차트로 표시"
  chartType="line"
  data={{
    points: [
      { x: 'Jan', y: 100 },
      { x: 'Feb', y: 150 },
      { x: 'Mar', y: 120 },
      { x: 'Apr', y: 180 },
      { x: 'May', y: 200 },
      { x: 'Jun', y: 170 },
    ],
  }}
  config={{
    width: 800,
    height: 400,
    colors: ['#3366ff'],
    showGrid: true,
  }}
  code={`import { createChart } from '@xelstack/chart-core';

const container = document.getElementById('chart')!;

const dataset = {
  points: [
    { x: 'Jan', y: 100 },
    { x: 'Feb', y: 150 },
    { x: 'Mar', y: 120 },
    { x: 'Apr', y: 180 },
    { x: 'May', y: 200 },
    { x: 'Jun', y: 170 },
  ],
};

const chart = createChart(container, dataset, {
  type: 'line',
  width: 800,
  height: 400,
  colors: ['#3366ff'],
  showGrid: true,
});`}
  client:load
/>

### 다중 시리즈 선 차트
