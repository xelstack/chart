# Interactive Examples (인터랙션 예제)

사용자 상호작용이 포함된 차트 예제를 살펴보세요.

## 데이터 업데이트

### 버튼 클릭으로 데이터 업데이트

버튼을 클릭하면 차트 데이터가 업데이트됩니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const initialData = {
  points: [
    { x: 'Jan', y: 100 },
    { x: 'Feb', y: 150 },
    { x: 'Mar', y: 120 },
  ],
};

const chart = createChart(container, initialData, {
  type: 'line',
  colors: ['#3366ff'],
  showGrid: true,
});

// 데이터 업데이트 버튼
document.getElementById('update-btn')?.addEventListener('click', () => {
  const newData = {
    points: [
      { x: 'Jan', y: Math.random() * 200 },
      { x: 'Feb', y: Math.random() * 200 },
      { x: 'Mar', y: Math.random() * 200 },
    ],
  };

  chart.updateData(newData);
  chart.render();
});
```

### 실시간 데이터 추가

새로운 데이터 포인트를 계속 추가합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const chart = createChart(container, { points: [] }, {
  type: 'line',
  colors: ['#3366ff'],
  showGrid: true,
});

let time = 0;

document.getElementById('start-btn')?.addEventListener('click', () => {
  const interval = setInterval(() => {
    time++;

    const currentPoints = chart.getData().points;
    const newPoint = {
      x: time,
      y: Math.sin(time / 5) * 50 + 50,
    };

    // 최근 50개만 유지
    const updatedPoints = [...currentPoints, newPoint].slice(-50);

    chart.updateData({ points: updatedPoints });
    chart.render();
  }, 100);

  // 정지 버튼
  document.getElementById('stop-btn')?.addEventListener('click', () => {
    clearInterval(interval);
  });
});
```

## 설정 변경

### 차트 타입 전환

드롭다운으로 차트 타입을 변경합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const dataset = {
  points: [
    { x: 'A', y: 100 },
    { x: 'B', y: 150 },
    { x: 'C', y: 120 },
    { x: 'D', y: 180 },
  ],
};

let chart = createChart(container, dataset, {
  type: 'line',
  colors: ['#3366ff'],
  showGrid: true,
});

// 차트 타입 선택
document.getElementById('chart-type')?.addEventListener('change', (e) => {
  const newType = (e.target as HTMLSelectElement).value as
    | 'line'
    | 'bar'
    | 'pie'
    | 'scatter';

  // 기존 차트 제거
  chart.destroy();

  // 새 차트 생성
  chart = createChart(container, dataset, {
    type: newType,
    colors: ['#3366ff'],
    showGrid: newType !== 'pie',
  });
});
```

## 반응형 동작

### 윈도우 리사이즈

화면 크기가 변경되면 차트도 자동 조정됩니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const container = document.getElementById('chart')!;

const chart = createChart(container, dataset, {
  type: 'line',
  width: container.clientWidth,
  height: 400,
  colors: ['#3366ff'],
  showGrid: true,
});

// 리사이즈 핸들러 (디바운스 적용)
let resizeTimeout: ReturnType<typeof setTimeout>;

window.addEventListener('resize', () => {
  clearTimeout(resizeTimeout);

  resizeTimeout = setTimeout(() => {
    chart.updateConfig({
      width: container.clientWidth,
    });
    chart.render();
  }, 250); // 250ms 디바운스
});
```

### 전체화면 모드

전체화면으로 차트를 확대합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const container = document.getElementById('chart')!;

const chart = createChart(container, dataset, {
  type: 'line',
  width: container.clientWidth,
  height: 400,
  colors: ['#3366ff'],
  showGrid: true,
});

// 전체화면 버튼
document.getElementById('fullscreen-btn')?.addEventListener('click', async () => {
  if (!document.fullscreenElement) {
    await container.requestFullscreen();

    // 전체화면에 맞춰 크기 조정
    chart.updateConfig({
      width: window.innerWidth,
      height: window.innerHeight,
    });
    chart.render();
  } else {
    await document.exitFullscreen();

    // 원래 크기로 복원
    chart.updateConfig({
      width: container.clientWidth,
      height: 400,
    });
    chart.render();
  }
});
```

## 데이터 필터링

### 날짜 범위 필터

슬라이더로 날짜 범위를 선택합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

// 전체 데이터
const fullData = {
  points: Array.from({ length: 100 }, (_, i) => ({
    x: i,
    y: Math.sin(i / 10) * 50 + 100,
  })),
};

const chart = createChart(container, fullData, {
  type: 'line',
  colors: ['#3366ff'],
  showGrid: true,
});

// 범위 슬라이더
const startSlider = document.getElementById('start') as HTMLInputElement;
const endSlider = document.getElementById('end') as HTMLInputElement;

const updateRange = () => {
  const start = parseInt(startSlider.value);
  const end = parseInt(endSlider.value);

  // 선택된 범위만 표시
  const filteredPoints = fullData.points.slice(start, end);

  chart.updateData({ points: filteredPoints });
  chart.render();
};

startSlider?.addEventListener('input', updateRange);
endSlider?.addEventListener('input', updateRange);
```

### 시리즈 선택

체크박스로 표시할 시리즈를 선택합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

// 전체 데이터 (여러 시리즈)
const fullData = {
  points: [
    { x: 'Jan', y: 100, series: 'Product A' },
    { x: 'Feb', y: 150, series: 'Product A' },
    { x: 'Mar', y: 120, series: 'Product A' },
    { x: 'Jan', y: 80, series: 'Product B' },
    { x: 'Feb', y: 130, series: 'Product B' },
    { x: 'Mar', y: 110, series: 'Product B' },
    { x: 'Jan', y: 90, series: 'Product C' },
    { x: 'Feb', y: 140, series: 'Product C' },
    { x: 'Mar', y: 100, series: 'Product C' },
  ],
};

const chart = createChart(container, fullData, {
  type: 'line',
  colors: ['#3366ff', '#ff6633', '#33ff66'],
  showGrid: true,
});

// 체크박스 변경 이벤트
document.querySelectorAll('input[name="series"]').forEach((checkbox) => {
  checkbox.addEventListener('change', () => {
    // 선택된 시리즈만 필터링
    const selected = Array.from(
      document.querySelectorAll('input[name="series"]:checked')
    ).map((cb) => (cb as HTMLInputElement).value);

    const filteredPoints = fullData.points.filter((point) =>
      selected.includes(point.series)
    );

    chart.updateData({ points: filteredPoints });
    chart.render();
  });
});
```

## 애니메이션 효과

### 순차적 렌더링

데이터 포인트를 하나씩 추가하며 표시합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const fullData = [
  { x: 'Jan', y: 100 },
  { x: 'Feb', y: 150 },
  { x: 'Mar', y: 120 },
  { x: 'Apr', y: 180 },
  { x: 'May', y: 200 },
];

const chart = createChart(container, { points: [] }, {
  type: 'line',
  colors: ['#3366ff'],
  showGrid: true,
});

// 순차적으로 포인트 추가
let currentIndex = 0;

const addNextPoint = () => {
  if (currentIndex < fullData.length) {
    const currentData = chart.getData();
    const newPoints = [...currentData.points, fullData[currentIndex]];

    chart.updateData({ points: newPoints });
    chart.render();

    currentIndex++;
    setTimeout(addNextPoint, 500); // 500ms 간격
  }
};

// 시작 버튼
document.getElementById('animate-btn')?.addEventListener('click', () => {
  currentIndex = 0;
  chart.updateData({ points: [] });
  chart.render();
  addNextPoint();
});
```

### 값 전환 애니메이션

값이 변경될 때 부드럽게 전환합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const chart = createChart(container, dataset, {
  type: 'bar',
  colors: ['#3366ff'],
  showGrid: true,
});

// 값 애니메이션 함수
const animateValue = (
  from: number,
  to: number,
  duration: number,
  onUpdate: (value: number) => void
) => {
  const startTime = Date.now();

  const animate = () => {
    const elapsed = Date.now() - startTime;
    const progress = Math.min(elapsed / duration, 1);

    // Ease-out 효과
    const eased = 1 - Math.pow(1 - progress, 3);
    const current = from + (to - from) * eased;

    onUpdate(current);

    if (progress < 1) {
      requestAnimationFrame(animate);
    }
  };

  requestAnimationFrame(animate);
};

// 업데이트 버튼
document.getElementById('update-animated')?.addEventListener('click', () => {
  const currentData = chart.getData();

  // 각 포인트를 애니메이션으로 업데이트
  currentData.points.forEach((point, i) => {
    const newValue = Math.random() * 200;

    animateValue(point.y, newValue, 1000, (value) => {
      currentData.points[i].y = value;
      chart.updateData(currentData);
      chart.render();
    });
  });
});
```

## CSV 데이터 import

### 파일 업로드

CSV 파일을 업로드하여 차트를 생성합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const fileInput = document.getElementById('csv-file') as HTMLInputElement;

fileInput?.addEventListener('change', (e) => {
  const file = (e.target as HTMLInputElement).files?.[0];
  if (!file) return;

  const reader = new FileReader();

  reader.onload = (event) => {
    const csvText = event.target?.result as string;

    // CSV 파싱
    const lines = csvText.split('\n');
    const points = [];

    // 헤더 스킵
    for (let i = 1; i < lines.length; i++) {
      const [x, y] = lines[i].split(',');
      if (x !== undefined && x !== '' && y !== undefined && y !== '') {
        points.push({
          x: isNaN(Number(x)) ? x.trim() : Number(x),
          y: Number(y),
        });
      }
    }

    // 차트 생성 또는 업데이트
    if (chart) {
      chart.updateData({ points });
      chart.render();
    } else {
      chart = createChart(container, { points }, {
        type: 'line',
        colors: ['#3366ff'],
        showGrid: true,
      });
    }
  };

  reader.readAsText(file);
});
```

## 공유 및 내보내기

### URL로 공유

현재 차트 상태를 URL로 공유합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const chart = createChart(container, dataset, config);

// 공유 버튼
document.getElementById('share-btn')?.addEventListener('click', async () => {
  // 현재 상태를 JSON으로 직렬화
  const state = {
    data: chart.getData(),
    config: chart.getConfig(),
  };

  // Base64 인코딩
  const encoded = btoa(JSON.stringify(state));

  // 공유 URL 생성
  const shareUrl = `${window.location.origin}${window.location.pathname}?state=${encoded}`;

  // 클립보드에 복사
  try {
    await navigator.clipboard.writeText(shareUrl);
    alert('공유 URL이 클립보드에 복사되었습니다!');
  } catch (err) {
    console.error('Failed to copy URL:', err);
  }
});

// 페이지 로드 시 URL에서 상태 복원
const urlParams = new URLSearchParams(window.location.search);
const stateParam = urlParams.get('state');

if (stateParam !== null && stateParam !== '') {
  try {
    const state = JSON.parse(atob(stateParam));
    const chart = createChart(container, state.data, state.config);
  } catch (err) {
    console.error('Failed to restore state:', err);
  }
}
```

### 이미지로 내보내기

차트를 PNG 이미지로 다운로드합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const chart = createChart(container, dataset, config);

// 다운로드 버튼
document.getElementById('download-btn')?.addEventListener('click', () => {
  // Canvas 요소 가져오기
  const canvas = container.querySelector('canvas');
  if (!canvas) return;

  // 이미지로 변환
  canvas.toBlob((blob) => {
    if (!blob) return;

    // 다운로드
    const url = URL.createObjectURL(blob);
    const link = document.createElement('a');
    link.href = url;
    link.download = 'chart.png';
    link.click();

    URL.revokeObjectURL(url);
  });
});
```
