# 시작하기

XelStack Chart는 함수형 프로그래밍 패러다임으로 구축된 고성능 차트 라이브러리입니다. 이 가이드를 따라 5분 안에 첫 번째 차트를 만들어보세요.

## 설치

XelStack Chart는 npm, pnpm, yarn을 통해 설치할 수 있습니다.

```bash
# npm
npm install @xelstack/chart-core

# pnpm
pnpm add @xelstack/chart-core

# yarn
yarn add @xelstack/chart-core
```

## 첫 번째 차트 만들기

### 1. HTML 컨테이너 준비

차트를 렌더링할 HTML 요소를 준비합니다.

```html
<div id="chart-container" style="width: 800px; height: 600px;"></div>
```

### 2. 데이터 준비

차트에 표시할 데이터를 준비합니다.

```typescript
const dataset = {
  points: [
    { x: 0, y: 10 },
    { x: 1, y: 20 },
    { x: 2, y: 15 },
    { x: 3, y: 25 },
    { x: 4, y: 22 },
    { x: 5, y: 30 },
  ],
};
```

### 3. 차트 생성

`createChart` 함수를 사용하여 차트를 생성합니다.

```typescript
import { createChart } from '@xelstack/chart-core';

const container = document.getElementById('chart-container')!;

const chart = createChart(container, dataset, {
  type: 'line',
  title: '내 첫 번째 차트',
  width: 800,
  height: 600,
  colors: ['#3366ff'],
  showGrid: true,
});
```

### 4. 렌더링

차트가 자동으로 렌더링됩니다. 데이터를 업데이트하려면:

```typescript
// 데이터 업데이트
chart.updateData({
  points: [
    { x: 0, y: 15 },
    { x: 1, y: 25 },
    { x: 2, y: 20 },
    { x: 3, y: 30 },
    { x: 4, y: 27 },
    { x: 5, y: 35 },
  ],
});

// 렌더링
chart.render();
```

## 전체 예제

다음은 완전한 예제 코드입니다:

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>XelStack Chart 예제</title>
</head>
<body>
  <div id="chart-container" style="width: 800px; height: 600px;"></div>

  <script type="module">
    import { createChart } from '@xelstack/chart-core';

    const container = document.getElementById('chart-container');

    const dataset = {
      points: [
        { x: 0, y: 10 },
        { x: 1, y: 20 },
        { x: 2, y: 15 },
        { x: 3, y: 25 },
        { x: 4, y: 22 },
        { x: 5, y: 30 },
      ],
};

    const chart = createChart(container, dataset, {
      type: 'line',
      title: '매출 추이',
      width: 800,
      height: 600,
      colors: ['#3366ff'],
      showGrid: true,
    });
  </script>
</body>
</html>
```

## 주요 개념

### 함수형 API

XelStack Chart는 함수형 프로그래밍 패러다임을 따릅니다:

- **순수 함수**: 동일한 입력에 대해 항상 동일한 출력
- **불변성**: 원본 데이터를 변경하지 않음
- **타입 안전성**: TypeScript로 작성되어 완벽한 타입 지원

### 차트 핸들

`createChart` 함수는 차트를 제어할 수 있는 핸들을 반환합니다:

```typescript
interface ChartHandle {
  updateData(dataset: Dataset): void;
  updateConfig(config: Partial<ChartConfig>): void;
  render(): void;
  destroy(): void;
}
```

### 고성능

- 10,000개 이상의 데이터 포인트 지원
- 자동 가상화 및 샘플링
- 저사양 디바이스 자동 감지 및 최적화
