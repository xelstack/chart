# 빠른 시작

XelStack Chart로 첫 번째 차트를 5분 안에 만들어보세요.

## 기본 설정

먼저 패키지를 설치합니다:

```bash
npm install @xelstack/chart-core
```

## 첫 번째 차트 만들기

### 1. HTML 준비

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First Chart</title>
</head>
<body>
  <div id="chart-container" style="width: 800px; height: 400px;"></div>
  <script type="module" src="./main.js"></script>
</body>
</html>
```

### 2. JavaScript 코드

```javascript
import { createChart } from '@xelstack/chart-core';

// 차트를 렌더링할 컨테이너 찾기
const container = document.getElementById('chart-container');

// 데이터 준비
const dataset = {
  points: [
    { x: 1, y: 10 },
    { x: 2, y: 25 },
    { x: 3, y: 17 },
    { x: 4, y: 32 },
    { x: 5, y: 28 }
  ]
};

// 차트 설정
const config = {
  type: 'line',
  title: 'My First Chart',
  width: 800,
  height: 400,
  showGrid: true
};

// 차트 생성
const chart = createChart(container, dataset, config);
```

### 3. 결과 확인

브라우저에서 HTML 파일을 열면 선 차트가 표시됩니다!

## 다른 차트 타입 시도하기

### 막대 차트

```javascript
const config = {
  type: 'bar',
  title: 'Bar Chart Example',
  colors: ['#3366ff', '#ff3366']

```

```javascript

```

## 차트 업데이트하기

생성된 차트는 ChartHandle을 통해 제어할 수 있습니다:

```javascript
// 데이터 업데이트
chart.updateData({
  points: [
    { x: 1, y: 15 },
    { x: 2, y: 30 },
    { x: 3, y: 22 }
  ]
});

// 설정 업데이트
chart.updateConfig({
  title: 'Updated Chart',
  colors: ['#ff6633']
});

// 수동 렌더링 (필요한 경우)
chart.render();
```

## 실시간 데이터 업데이트

실시간으로 업데이트되는 차트를 만들어보세요:

```javascript
const chart = createChart(container, dataset, config);

// 1초마다 새로운 데이터 포인트 추가
setInterval(() => {
  const currentData = chart.getData();
  const newPoint = {
    x: currentData.points.length + 1,
    y: Math.random() * 50
  };

  chart.updateData({
    points: [...currentData.points, newPoint]
  });
  chart.render();
}, 1000);
```

## 일반적인 문제 해결

### 차트가 표시되지 않는 경우

1. 컨테이너 요소가 존재하는지 확인
2. 컨테이너에 너비와 높이가 설정되어 있는지 확인
3. 브라우저 콘솔에서 에러 메시지 확인

### TypeScript 사용 시

```typescript
import { createChart, type Dataset, type ChartConfig } from '@xelstack/chart-core';

const dataset: Dataset = {
  points: [
    { x: 1, y: 10 },
    { x: 2, y: 25 }
  ]
};

const config: ChartConfig = {
  type: 'line',
  title: 'Typed Chart'
};

const chart = createChart(container, dataset, config);
```
